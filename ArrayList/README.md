# Java ArrayList Practice 🚀

A structured, level-wise collection of Java programs to master the `ArrayList` class — from basics to placement/interview-level problems. This is the starting point of my **Java Collections Framework** practice series.

Each level is implemented in its **own file**, progressing from fundamentals to advanced problem-solving.

## 📌 Roadmap

### ✅ Level 1: ArrayList Basics (Easy)
1. Create an ArrayList and add 10 integers.
2. Print all elements using `for` loop, enhanced `for` loop, and `Iterator`.
3. Find the size of an ArrayList.
4. Get an element at a given index.
5. Update an element using `set()`.
6. Remove an element by index.
7. Remove an element by value.
8. Check whether an element exists using `contains()`.
9. Find the index of an element.
10. Clear the ArrayList.

### ✅ Level 2: Basic Logic
1. Find the sum of all elements.
2. Find the average.
3. Find the largest element.
4. Find the smallest element.
5. Count even numbers.
6. Count odd numbers.
7. Reverse an ArrayList without using `Collections.reverse()`.
8. Reverse using `Collections.reverse()`.
9. Print elements in reverse order.
10. Copy one ArrayList into another.

### ✅ Level 3: Intermediate Problems
1. Find the second largest element.
2. Find the second smallest element.
3. Remove duplicate elements.
4. Find the frequency of each element.
5. Find the most frequent element.
6. Find the least frequent element.
7. Rotate the ArrayList by one position.
8. Rotate by k positions.
9. Left rotate by k positions.
10. Right rotate by k positions.

### ✅ Level 4: Searching
1. Linear Search.
2. Binary Search (sorted ArrayList).
3. Find first occurrence.
4. Find last occurrence.
5. Count occurrences of an element.

### ✅ Level 5: Sorting
1. Sort in ascending order.
2. Sort in descending order.
3. Sort strings alphabetically.
4. Sort by string length.
5. Sort custom objects (`Student`) by age.

### ✅ Level 6: ArrayList + Strings
1. Store names and print them.
2. Find the longest string.
3. Find the shortest string.
4. Remove all strings starting with a vowel.
5. Convert all strings to uppercase.
6. Count words having more than 5 letters.
7. Find duplicate strings.

### ✅ Level 7: ArrayList Methods Practice
Practicing each method individually:
`add()`, `add(index, value)`, `addAll()`, `remove()`, `removeAll()`, `retainAll()`, `contains()`, `containsAll()`, `indexOf()`, `lastIndexOf()`, `set()`, `get()`, `clear()`, `isEmpty()`, `size()`, `toArray()`, `subList()`, `clone()`

### ✅ Level 8: Collections Utility Class
1. `Collections.sort()`
2. `Collections.reverse()`
3. `Collections.shuffle()`
4. `Collections.max()`
5. `Collections.min()`
6. `Collections.frequency()`
7. `Collections.swap()`
8. `Collections.fill()`
9. `Collections.binarySearch()`

### ✅ Level 9: Two ArrayLists
1. Merge two ArrayLists.
2. Find common elements.
3. Find union.
4. Find intersection.
5. Find difference.
6. Check if two ArrayLists are equal.
7. Remove common elements.

### ✅ Level 10: ArrayList of Objects
Using a `Student` class (`id`, `name`, `marks`):
1. Add 10 students.
2. Print all students.
3. Search by ID.
4. Search by name.
5. Find topper.
6. Find average marks.
7. Sort by marks.
8. Sort by name.
9. Remove a student by ID.
10. Update marks.
11. Count students scoring above 75.

### ✅ Level 11: Placement-Level Problems
1. Move all zeros to the end.
2. Remove all even numbers.
3. Find missing number (1 to n).
4. Find duplicate number.
5. Find all duplicates.
6. Merge two sorted ArrayLists.
7. Remove duplicates from a sorted ArrayList.
8. Find pair with given sum.
9. Find triplet with given sum.
10. Leaders in an ArrayList.
11. Majority element.
12. Kadane's Algorithm (Maximum Subarray Sum).
13. Product of array except self.
14. Next Greater Element.
15. Rearrange positive and negative numbers.

### ✅ Level 12: Advanced (Interview)
1. Spiral matrix using `ArrayList<ArrayList<Integer>>`
2. Matrix transpose.
3. Matrix multiplication.
4. Rotate matrix by 90°.
5. Pascal's Triangle.
6. Flatten a 2D ArrayList.
7. Group anagrams.
8. Top K frequent elements.
9. Sliding window maximum.
10. Merge intervals.

## ⚙️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/saarthak1911/ArrayList-Practice.git
   ```
2. Navigate into the folder:
   ```bash
   cd ArrayList-Practice
   ```
3. Compile and run any level file:
   ```bash
   javac Level1_Basics.java
   java Level1_Basics
   ```
